# ✈️ Trrip — AI Travel Itinerary Generator

A MERN-based web application that automatically generates AI-powered travel itineraries from uploaded booking documents.

## Features
- JWT Authentication (Login/Register)
- Upload flight tickets, hotel bookings (PDF/Images)
- AI-powered itinerary generation using Gemini API
- View itinerary history
- Shareable itinerary links
- Drag & drop file uploads
- Responsive UI

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **AI:** Google Gemini API
- **Deployment:** Render (backend) + Netlify (frontend)

## Live Demo
- Frontend: https://trrip-itinerary-app.netlify.app/login
- Backend: https://trrip-backend-enwe.onrender.com/

## Setup
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
